Five HTML Tags

The heading tag
The h tag is a code that transforms normal text into a larger and bold version, to show it's importance. It creates emphasis and seperates sections in a clean manner. The headings range from 1-6, however, h1 must only be on a page once, whereas every other number following can be repeated. This is a paired tag.
The footer tag
The footer tag seperates content that should be placed at the top or center of the page, and pushes it to the end. This is done by creating a section for the information usually listed at the end such as: contact us and copyright information. This is a paired tag.
The image tag
The img tag impliments photographs or images into the page. It can be put inside the code either through the relative or absolute file path, with relative sourcing it from the subfolders within the main code folder, and and absolute sourcing it from the internet. When using the image tag, one must insert an alt, which is a brief descritption of the image. An alt is not seen unless the image, for some reason, is not working. It is also usefull for text to speech for the visuall impaired. This is a singular/empty tag.
The paragraph tag
The p tag is used to identify a paragraph in the code. It ensures readability of the text in the code which the coder wants displayed on the page. This is a paired tag.
The description list tag
The dl tag creates a list which features subjects to be described. This is the tag currently being used. This tag must have a dt and a dd tag within the group. The dt tag is the general name of the work(s) being described. The dd tag is used to describe the term in the dt tag above it. This is a paired tag.
